## Description
**RAM Killer** is a high-speed tool designed to aggressively consume system memory. It is intended **only for testing and educational purposes** in controlled environments. Running this program on critical or production systems can result in immediate system instability or crashes.

This tool is available in **two versions**: C# and Python.  

### C# Version
- Compiled executable (`.exe`), requires .NET Framework or .NET 5+.
- High performance; consumes RAM faster than Python.
- Works primarily on Windows systems.
- More optimized memory management.
- Distribution is simple as a single executable.

**Pros:** Fast, reliable, easy Windows deployment.  
**Cons:** Windows-focused, code not easily modifiable.

### Python Version
- Scripted, runs via Python interpreter (Python 3.x recommended).
- Slightly slower in memory consumption compared to C#.
- Platform-independent: works on Windows, Linux, and Mac.
- Code is open and easily editable for testing or learning purposes.

**Pros:** Cross-platform, easy to modify, ideal for educational use.  
**Cons:** Slower than C#, requires Python installed.
